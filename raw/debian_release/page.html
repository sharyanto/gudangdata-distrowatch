The Debian Project is an association of individuals who have made common cause to create a free operating system. This operating system is called Debian. Debian systems currently use the Linux kernel. Linux is a completely free piece of software started by Linus Torvalds and supported by thousands of programmers worldwide. Of course, the thing that people want is application software: programs to help them get what they want to do done, from editing documents to running a business to playing games to writing more software. Debian comes with over 50,000 packages (precompiled software that is bundled up in a nice format for easy installation on your machine) - all of it free. It's a bit like a tower. At the base is the kernel. On top of that are all the basic tools. Next is all the software that you run on the computer. At the top of the tower is Debian -- carefully organizing and fitting everything so it all works together.

Great! It works for me!

I install it to the hard disk, and always the installer failed the Broadcom wifi card configuration, complaining that it needed non-free software. Then why did the live session work with the card?

I eliminated the Broadcom card from the system and used a client adapter hooked in by my system onboard Ethernet (my onboard Ethernet is always configured correctly). I retested, reinstalled and the installation to hard disk began to refuse to configure the ATI card. Then why did the ATI card get configured during the live session?

Still, I've used many 'buntu derivatives, I know the Debian base can work well for me. Just gotta figure out how to repeat the live session configuration when I install to hard disk!!

I have tried Debian based distros like- ubuntu, sparky, MX linux etc. The only difference between them and debian was not containing non-free contrib repo default by Debian. You can include non-free repo in etc/apt/sources.list. Just because a repo it shouldn't be called non-friendly distro. The installation in gtk mode was user-friendly. I connected my network during installation and everything went well. If you used live cd for installation, you would call it a user-friendly distro.

2. Lightweight- I would say here that Debian is the most lightweight linux distro of the world. You will ask my why ? I would reply you Ubuntu which is debian based distro is full of crap. Have you ever checked and compared free RAM of both distros with same desktop environment. Well I have checked it with free -m command in terminal. All tests were run on a 32bit-processor and 1GB RAM computer. All three distros were updated and fresh installed. None of the program's tab was opened except of xfce terminal.
1 Debian XFCE 9.9- 128MB (The default themes were ugly but later I installed arc style and moka icons themes and checked free -m again. The result was 152MB)
2. Xubuntu 18.04 - 302MB
3. MX Linux 18.3- 262MB
I checked Lubuntu vs debian LXDE on same machine as well. The results were-
Debian LXDE 9.9- 108MB
Lubuntu 18.4- 262MB

3. Stable- Debian is the most stable distro among all. After getting a horrible update on my previous lubuntu OS, my icons were disappeared and themes got a cross icon sign. It has been for 2 months I'm using debian but it doesn't has any bug so far.
4. An independent distro - 125 out 289 distos (on the distrowatch.com) are based on debian itself. It is not a fork. A fork distro can not be as stable as an independent distro can be.
