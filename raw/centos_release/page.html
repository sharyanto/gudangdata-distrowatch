CentOS

Last Update: 2020-12-25 15:51 UTC

  • OS Type: Linux
  • Based on: Fedora, Red Hat
  • Origin: USA
  • Architecture: aarch64, ppc64le, x86_64
  • Desktop: GNOME, KDE
  • Category: Desktop, Live Medium, Server
  • Status: Active
  • Popularity: 20 (517 hits per day)
CentOS as a group is a community of open source contributors and users which started in 2003 and has been sponsored by Red Hat since 2014. CentOS Linux versions up to CentOS Linux 8 are 100% compatible rebuilds of Red Hat Enterprise Linux, in full compliance with Red Hat's redistribution requirements. In 2020 it was announced CentOS Linux is being discontinued and replaced with CentOS Stream, a developer-focused distribution which acts as a middle-stream between Fedora and Red Hat Enterprise Linux.

 After working for many years with CentOS (from CentOS 5 to CentOS 7) I moved on to other distros, but CentOS still remains the best for stability and use. I must say that I miss working with CentOS.
CentOS intends to be an enterprise-type server distro and I would say that they have achieved the goal.

It makes no sense for this distro to discuss desktop or user functionality, or even the graphical interface, as in my personal vision CentOS should be used with minimal installation and SSH interface. Despite this, I have to say a few words.
There is no doubt that being able to install a light and intuitive graphical interface, the management of the server is more comfortable, but in the latest version, the too minimal interface of the desktop leaves me very perplexed. I prefer a more standard interface (IMHO), more similar to windows because the ergonomics of use improve.

As for the server-oriented use, unfortunately, the conservative policy does not make it suitable even for the management of commercial web servers due to the backwardness of updates on PHP and other environments. When a platform version update forces you to update PHP (perhaps for security reasons), and CentOS doesn't make it available for a long time, you don't become particularly satisfied. Therefore for use as a webserver you need to add external repos, which sometimes make CentOS less stable and a more negative impression in people's heads.
For use as a DB server, firewall and other types of services that do not require technological updates, it is the best linux distro.
